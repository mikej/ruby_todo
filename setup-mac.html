Instructions for getting set up for the workshop on a Mac

Download code View on GitHub

If you're running Yosemite (OS X 10.10) or Mavericks (OS X 10.9) then you'll already have a new enough version of Ruby to take part in the workshop. You just need to install the developer tools that are needed for some of the add-on code we'll be using. To do this:

  1. Open the Applications folder in Finder, open the Utilities folder inside there, and then open the Terminal app
  2. type the command gcc and press Enter
    1. if the developer tools are already installed you'll see a message `..sdfsd` and you're all set
    2. if not, you'll see a prompt "click install command line"..
