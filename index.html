Introduction

In this workshop we're going to make a simple todo list app that will allow adding items to a list, marking them as done, and maybe more if time allows! Along the way we'll write some Ruby and JavaScript code and will cover getting started with the Sinatra web framework, SQL database queries and some jQuery.

To get the most out of this session it will be useful to have some familiarity with writing web pages using HTML and CSS and to have picked up a few programming basics such as using variables, conditions, loops and writing your own functions in JavaScript, Ruby or a similar language.

To save time on the day it would be really handy to get your laptop set up using the instructions below.
