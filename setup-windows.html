Instructions for getting set up for the workshop on Windows

Download code View on GitHub

Download RubyInstaller and Development Kit

For 32-bit Windows: For 32 bit Windows:

  • rubyinstaller-2.1.6.exe
  • DevKit-mingw64-32-4.7.2-20130224-1151-sfx.exe

For 64-bit Windows:

  • rubyinstaller-2.1.6-x64.exe
  • DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe

If unsure which you need, use the 32-bit versions.

Run RubyInstaller Setup

Run the downloaded rubyinstaller, follow the steps:

  1. Select the language to use during the installation
  2. accept the licence agreement
  3. choose destination and optional tasks (be sure to tick *add Ruby executables to your PATH*)
  4. click *Finish*

Install Development Kit

Run the downloaded DevKit-mingw...exe

  1. enter a folder to extract the files e.g. `C:\RubyDevKit`, click *Extract*
  2. open a Command Prompt
  3. CD into the directory where the files were extracted e.g. cd \RubyDevKit
  4. ruby dk.rb init
  5. ruby dk.rb review
  6. ruby dk.rb install

See Wiki for more details: https://github.com/oneclick/rubyinstaller/wiki/Development-Kit

Install Bundler

  1. gem install bundler
  2. click Allow access if prompted by Windows Firewall

Download the starting point code for the workshop

  1. Download the starting point zip and unzip it
  2. bundle install
  3. Download the SQLite DLL:
    1. http://mirror.nienbo.com/sqlite/sqlite-dll-win32-x86-3071700.zip
    2. unzip this and copy DLL into ruby bin
  4. Download SQLite shell:
    1. http://mirror.nienbo.com/sqlite/sqlite-shell-win32-x86-3071700.zip
    2. unzip this and copy DLL into ruby bin
